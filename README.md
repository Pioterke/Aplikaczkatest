# HTML-App-Builder
Fork this repository & add any changes you'd like. After updating, an APK should be in your Actions tab & whatnot. You need basic knowledge of Github to use this.

You shoupd edit capacitor.config.json, index.html, style.css and package.json for customizability. The folder "resources" is for the splash screen & app icon, so feel free to add your own splash screen & icons. Beware, name the icon "icon.png" and the splash screen "splash.png" so that it works. Otherwise, you'll need to edit build.yml.
